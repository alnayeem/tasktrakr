# TaskTrakr UI Screens & Flows

> Load after: [00-MVP-PLAN.md](./00-MVP-PLAN.md)

## Screen List (MVP)

1. **Onboarding** - Language selection, welcome
2. **Home/Dashboard** - Today's tasks, active goals
3. **Goal Creation** - Input goal, select duration
4. **AI Loading** - Generating plan animation
5. **Goal Detail** - Task list, progress, streaks
6. **Settings** - Language, theme, about

---

## User Flows

### First-Time User
```
App Open â†’ Language Selection â†’ Welcome â†’ Create Goal â†’ AI Loading â†’ Dashboard
```

### Returning User
```
App Open â†’ Dashboard (Today's Tasks) â†’ Check off tasks â†’ See progress
```

### Create New Goal
```
Dashboard â†’ "+" Button â†’ Enter Goal â†’ Select Duration â†’ (Optional: Category) â†’ Generate â†’ View Plan
```

## RTL Support (Arabic)

When language is Arabic:
- All layouts flip horizontally
- Text aligns right
- Navigation arrows reverse
- Use `Directionality` widget in Flutter

```dart
Directionality(
  textDirection: isArabic ? TextDirection.rtl : TextDirection.ltr,
  child: // your widget
)
```
